# AI Prompts Used During Development

This document contains the major AI prompts used while developing the **Cash Flow – Salary & Expense Tracker** project.

---

# Prompt 1 – Project Setup

Create a modern and responsive **Cash Flow – Salary & Expense Tracker** using HTML5, CSS3, and Vanilla JavaScript.

Requirements:

* Clean folder structure
* Responsive layout
* Semantic HTML
* Professional UI
* No React or Bootstrap

---

# Prompt 2 – User Interface

Build a responsive dashboard containing:

* Header
* Salary Input Section
* Expense Form
* Summary Cards
* Expense List
* Analytics Section
* Report Section

Use CSS Grid and Flexbox.

---

# Prompt 3 – Salary Module

Implement the salary management module.

Requirements:

* Save monthly salary
* Validate input
* Prevent negative values
* Display salary in summary cards
* Update remaining balance automatically

---

# Prompt 4 – Expense Module

Implement expense management.

Requirements:

* Add new expense
* Validate inputs
* Store expenses in an array
* Render expenses dynamically
* Clear form after submission

---

# Prompt 5 – Remaining Balance Calculation

Implement automatic calculations.

Formula:

Remaining Balance = Total Salary − Total Expenses

Update:

* Total Expenses
* Remaining Balance
* Summary Cards

without refreshing the page.

---

# Prompt 6 – Input Validation

Create reusable validation functions.

Validate:

* Empty fields
* Negative values
* Zero values
* Invalid numbers
* Whitespace-only text

Display user-friendly error messages.

---

# Prompt 7 – Local Storage

Implement persistent storage.

Requirements:

* Save salary
* Save expenses
* Restore data after page refresh
* Handle corrupted localStorage safely

---

# Prompt 8 – Delete Expense

Implement expense deletion.

Requirements:

* Delete selected expense
* Update UI
* Update localStorage
* Recalculate balance
* Update chart

---

# Prompt 9 – Expense Analytics

Integrate Chart.js.

Requirements:

* Pie Chart
* Show Expenses vs Remaining Balance
* Auto-update chart whenever data changes
* Responsive design

---

# Prompt 10 – PDF Report

Implement PDF report generation using jsPDF.

Include:

* Current Date
* Salary
* Total Expenses
* Remaining Balance
* Expense List

Download as:

CashFlow_Report.pdf

---

# Prompt 11 – Currency Conversion

Implement currency conversion support.

Requirements:

* Currency selector
* Display values in selected currency
* Update salary, expenses and balance
* Maintain original values internally

---

# Prompt 12 – Low Balance Warning

Implement a warning system.

If Remaining Balance falls below 10% of salary:

* Display warning message
* Highlight remaining balance
* Improve user awareness

---

# Prompt 13 – Responsive Design

Improve responsiveness for:

* Mobile
* Tablet
* Laptop
* Desktop

Ensure proper spacing, alignment and typography.

---

# Prompt 14 – Accessibility

Improve accessibility.

Requirements:

* Semantic HTML
* Keyboard navigation
* Focus states
* Proper labels
* Accessible forms

---

# Prompt 15 – Final Review

Review the entire project.

Verify:

* UI
* Functionality
* Validation
* Local Storage
* Chart.js
* PDF Generation
* Responsiveness
* Accessibility
* Code Quality

Optimize the project without changing the existing functionality.
